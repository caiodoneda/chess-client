# Contributing

The main purpose of this repository is to continue to evolve the client, supporting more pieces, making it faster and easier to use.

Fell free to create issues and submit new Pull requests.

# Installation

- Clone the repository using `https://github.com/caiodoneda/chess-client.git` or download the ZIP file.
- `cd chess-client/`
- run the command `npm install` to install all dependencies.

# Env variables

Create a file `.env.local` based on `.env.local.example` and provide the values for the variables.
